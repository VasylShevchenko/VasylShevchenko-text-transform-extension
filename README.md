# Development

1. ng build
2. cp manifest.json dist/
3. open chrome and input: chrome://extensions/
4. enable 'Developer mode'
5. click 'Load unpacked extensionâ€¦' and choose path/to/the/extension/dist
6. click 'Enabled'
